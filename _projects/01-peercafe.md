## Overview

PeerCafe is a comprehensive restaurant ordering platform that connects customers with restaurants and enables seamless food ordering and delivery management.

## Features

### User Features
- **Restaurant Discovery**: Browse restaurants by cuisine, ratings, and location
- **Menu Management**: View detailed menus with item descriptions, prices, and availability
- **Shopping Cart**: Add/remove items with real-time cart updates
- **Order Tracking**: Monitor order status in real-time
- **User Profile**: Manage personal information and order history

### Admin Features
- **Restaurant Dashboard**: Manage restaurant details and operational status
- **Menu Management**: Create, update, and delete menu items
- **Order Management**: View and manage incoming orders
- **Analytics**: Track sales and popular items

## Technology Stack

### Frontend
- **Framework**: React with Next.js for SSR and SSG
- **Language**: TypeScript for type safety
- **Styling**: Tailwind CSS and Material-UI
- **State Management**: React Context API
- **Testing**: Jest and React Testing Library

### Backend
- **Runtime**: Node.js with Express/FastAPI
- **Database**: PostgreSQL with Supabase
- **Authentication**: JWT-based auth with Supabase
- **Deployment**: Docker containers

### Infrastructure
- **Hosting**: GitHub Pages for frontend, AWS/Heroku for backend
- **CI/CD**: GitHub Actions
- **Version Control**: Git and GitHub
