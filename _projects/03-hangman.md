## Overview

A classic Hangman word-guessing game implemented in Java with an interactive graphical user interface.

## Features

- **Interactive GUI**: User-friendly interface built with Swing
- **Word Database**: Extensive word list for varied gameplay
- **Game Logic**: Proper win/lose conditions and score tracking
- **Visual Feedback**: Display hangman drawing progress
- **Replay Functionality**: Play multiple rounds without restarting

## Implementation Highlights

- Object-oriented design with clear class separation
- Event handling for user interactions
- String manipulation and character comparison
- Game state management

## Technologies

- Java
- Swing (GUI)
- File I/O for word database

## Future Enhancements

- Difficulty levels
