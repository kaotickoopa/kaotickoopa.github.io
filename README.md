## ğŸš€ Getting Started

### Prerequisites

- Ruby 2.7 or higher
- Bundler (install with `gem install bundler`)
- Git

### Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/kaotickoopa/kaotickoopa.github.io.git
   cd kaotickoopa.github.io
   ```

2. **Install dependencies**:
   ```bash
   bundle install
   ```

3. **Serve locally**:
   ```bash
   bundle exec jekyll serve
   ```

4. **Open in browser**:
   Navigate to `http://localhost:4000`

## ğŸ“ Creating Content

### Adding a Blog Post

1. Create a new file in `_posts/` with naming convention: `YYYY-MM-DD-slug.md`
2. Add front matter:
   ```markdown
   ---
   layout: post
   title: "Your Post Title"
   date: 2024-11-05
   author: "Your Name"
   tags: ["tag1", "tag2"]
   ---
   
   Your content here...
   ```

### Adding a Project

1. Create a new file in `_projects/` with naming convention: `01-project-slug.md`
2. Add front matter:
   ```markdown
   ---
   layout: project
   title: "Project Title"
   slug: "project-slug"
   subtitle: "Project Subtitle"
   date: 2024-11-01
   technologies: ["React", "Node.js"]
   role: "Full-Stack Developer"
   link: "https://github.com/username/repo"
   ---
   
   Your content here...
   ```

### Adding a Page

1. Create a new Markdown file in the root directory
2. Add front matter:
   ```markdown
   ---
   layout: page
   title: "Page Title"
   permalink: /page-slug/
   ---
   
   Your content here...
   ```

## ğŸ¨ Customization

### Edit Site Configuration

Update `_config.yml` with your information:
```yaml

  name: "Your Name"
```

Edit the CSS variables in `assets/css/style.css`:
```css

    --primary-color: #2c3e50;
    --secondary-color: #3498db;
    --accent-color: #e74c3c;
    /* ... other variables ... */
}
```

### Modify Homepage

Edit `index.md` to customize the hero section, featured projects, skills, etc.

## ğŸ“¦ Deployment

### GitHub Pages Deployment

The site is automatically deployed when you push to the `main` (or master) branch:

1. Push your changes:
   ```bash
   git add .
   git commit -m "Update content"
   git push origin main
   ```

2. GitHub Actions will automatically build and deploy your site
3. Your site will be live at `https://username.github.io`

### Custom Domain

To use a custom domain:

1. Add a `CNAME` file to the root with your domain
2. Update DNS records with GitHub Pages instructions
3. Enable HTTPS in GitHub Pages settings

## ğŸ› Troubleshooting

### Site not building?
- Check Ruby version: `ruby --version`
- Clear Jekyll cache: `bundle exec jekyll clean`
- Check for YAML syntax errors in front matter

### CSS not updating locally?
- Clear browser cache (Ctrl+Shift+Delete)
- Stop and restart the Jekyll server
- Check for CSS syntax errors in `assets/css/style.css`

### Posts/Projects not appearing?
- Verify front matter format (correct indentation and colons)
- Check file naming: `YYYY-MM-DD-slug.md`
- Ensure date is not in the future
- Check collection configuration in `_config.yml`
